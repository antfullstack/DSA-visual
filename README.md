# DSA-visual

## What it is: 

DSA-visual is publically available github repo with implementations of various DSA concepts paired with visualizations written in the python Manim Library. 

## Tech stack: 
Python (vanilla 3.14) 
Manim (python library)

## How to contribute: 
1.Making manim visuals. 
2.Adding DSA concepts code implementations. 
3.Improving the efficiency or explanation of code. 
4.Improving the docs. 
5.Reporting issues. 

## AI Convention
**Note**: AI-generated code will be accepted in the visualizations, however, it is encouraged to keep DSA explanations and code human-made. No one is perfect, it is encouraged to contribute in keeping the repo in a dev to dev climate.
